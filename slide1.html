Verifying that deaths due to Covid cases
might not increase even if Covid cases increases

Dataset being used: (NY-times) Coronavirus (Covid-19) Data in the United States: Link

The primary data published on the link is the daily cumulative number of cases and deaths reported in each county(and not country) and state across the U.S. since the beginning of the pandemic.



I am using "Martini Narrative Visualization Structure". As part of this Narrative Visualization Project, I am focusing on the following aspects of data:

  • Number of Cases as a function of time
    Plotting a line chart to visualize number of cases as the time progressed

  • Number of Deaths as a function of time
    Plotting a line chart to visualize number of deaths as the time progressed

  • Fatality Rate as a function of time
    Plotting a line chart to visualize fatality as the time progressed

  • Plotting deaths with covid cases and fatality rate with covid cases
    To give a sense of comparison

A Final plot where user can dig more by applying filters and reading tooltips
